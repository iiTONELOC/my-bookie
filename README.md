**MyBookie** is an appointment setter Full Stack Web Application Featuring:

Front End

- JWT
- React.js
- Tailwindcss
- Tailstrap (a Tailwindcss component library)

Back End

- JWT
- Bcrypt
- Python Flask Server
- PyMongo (MongoDB driver for Python)
- MongoEngine (DOM: Document-Object Mapper)
